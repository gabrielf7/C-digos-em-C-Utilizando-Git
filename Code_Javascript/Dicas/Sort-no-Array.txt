Referência: https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Reference/Global_Objects/Array/sort

Sobre:
O método sort é usado para ordenação de array na ordem crescente.

Cuidados ao usar SORT:
- A complexidade do método sort() tem uma limitação, o array é ordenado de acordo com a pontuação de código Unicode de cada um dos caracteres, ou seja, de acordo com a conversão de cada elemento para string, que pode resultar em falhas de ordenação.

Exemplo de falha:
let numeros = [4, 79, 2, 22, 7];
numeros.sort();
console.log(numeros);
// [ 2, 22, 4, 7, 79 ] 

(Observação: Essa operação de ordenação está errada, note isso pelo número 22).

-------------------------------------------------------------------------------------------------------------------------------------------
Solução para números:
-------------------------------------------------------------------------------------------------------------------------------------------

let numeros = [4, 79, 2, 22, 7];
numeros.sort(function(a, b) {
  return a - b;
});
console.log(numeros);

-------------------------------------------------------------------------------------------------------------------------------------------
Solução para texto:
-------------------------------------------------------------------------------------------------------------------------------------------

let pessoas = [ { nome: 'Gabriel', idade: 25 }, { nome: 'Ana', idade: 27 }, { nome: 'Isabela', idade: 22 } ];
pessoas.sort(function (a, b) {
  if (a.nome > b.nome) {
    return 1;
  } else if (a.nome < b.nome) {
    return -1;
  }
  // "a" deve ser igual a "b"
  return 0;
});
console.log(pessoas);
